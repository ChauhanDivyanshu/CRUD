# 🚀 CRUD Application

A full-stack CRUD (Create, Read, Update, Delete) application built to demonstrate modern web development practices using the MERN Stack. This project enables users to create, view, update, and delete records through a clean and responsive user interface backed by RESTful APIs.

---

## 📌 Features

* Create new records
* View all records
* Update existing records
* Delete records
* RESTful API integration
* Responsive user interface
* Form validation
* Error handling
* MongoDB database connectivity
* Clean and modular project structure

## 🌐 API Operations

| Method | Endpoint         | Description               |
| ------ | ---------------- | ------------------------- |
| GET    | `/api/items`     | Fetch all records         |
| GET    | `/api/items/:id` | Fetch a single record     |
| POST   | `/api/items`     | Create a new record       |
| PUT    | `/api/items/:id` | Update an existing record |
| DELETE | `/api/items/:id` | Delete a record           |
